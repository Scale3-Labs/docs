Onboarding using the pull method offers a lightweight option for monitoring if you don't want to use the agent. 

Follow the steps below to add your node:

# How to setup monitoring

- Head over to [www.scale3labs.com](https://www.scale3labs.com/signup) and sign up for an account


<img height="200" src="https://miro.medium.com/v2/resize:fit:1400/1*JwrNViuYqbVjWDdraU-blw.png" />

- At the chain selection prompt, select one

<img height="200" src="https://miro.medium.com/v2/resize:fit:1400/1*so-EyDWyf1EeMAyy3gO_Lg.png" />

- For installation options, choose the IP/URL option.

<img height="200" src="https://miro.medium.com/v2/resize:fit:1400/1*bBX4HESWjrSADOVhNpfmXQ.png" />

- If adding a SUI node, add the IP/URL for your node. If adding an ETH node (Geth & Lighthouse), be sure to add the IP & URL for both Execution and Consensus Clients.

- That’s it! You can now click Go to Dashboard and wait for your dashboard to light up within 5 minutes. Be sure to select the node on the left panel to see your dashboard.

<img height="200" src="https://miro.medium.com/v2/resize:fit:1400/1*Y2-bjiHQpC4-CY-BYoGFFg.png" />
