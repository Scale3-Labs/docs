NodeKeeper is a powerful infrastructure product designed to simplify the setup and maintenance. It empowers users with full control over their servers, making it easy to develop, test, and deploy blockchain applications. Whether you are a blockchain enthusiast, developer, startup, or enterprise, NodeKeeper is designed to meet your unique needs.

# Who should use NodeKeeper?

NodeKeeper is designed for anyone who wants to run a blockchain node. Whether you are a blockchain enthusiast, developer, startup, or enterprise, NodeKeeper is designed to meet your unique needs.
It's for someone who wants to run a node but doesn't want to deal with the hassle of setting up and maintaining it.

# Use Cases

- **Build Blockchain Indexers**: Create powerful blockchain indexers for efficient data retrieval and analysis.

- **Develop Blockchain Explorers**: Build user-friendly blockchain explorers to visualize transaction data.

- **Blockchain Analysis**: Use NodeKeeper to facilitate blockchain intelligence tools for valuable insights.

- **Decentralized Application (dApp) Development**: Ensure smooth integration of dApps with reliable node management.

- **Blockchain Protocol Development**: Easily set up and manage blockchain nodes for development and testing.

# Key Features

- Automated Node Provisioning on any server
- One click node updates
- Seamless integration for other Scale3 products
