You can set up three different types of alerts:
- [Node health](/alerting/node-health) alerts such as:
  - CPU, memory, disk, and network usage
  - Peers and blocks
  - Node status
  - And many more!
- [Validator Onchain](/alerting/validator-alerts) alerts for validators such as:
  - Low stake alerts
  - Validator at risk of slashing
  - Reminder to set Reference Gas Price(RGP)
  - And many more!
- [Network intelligence](/alerting/network-intelligence) and [Node updates](/alerting/network-intelligence) alerts such as:
  - Important network events
  - Network upgrade notifications
  - Node updates
