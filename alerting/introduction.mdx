You can set up three different types of alerts:
- Node health alerts such as:
  - CPU, memory, disk, and network usage
  - Peers and blocks
  - Node status
- Validator metrics alerts such as:
  - Low stake alerts
  - Validator at risk of slashing
  - Reminder to set Reference Gas Price(RGP)
- Network intelligence and Node updates alerts such as:
  - Important network events
  - Network upgrade notifications
  - Node updates

  > **Note**: We support alert notifications via email, Slack, Pagerduty and FireHydrant. Refer to the [Channels](/channels/introduction) section for more details.
