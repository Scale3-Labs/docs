## Updating The Sui Fullnode Binary

If you are running a Sui full node or a Sui validator, you can use Sui Wizard to update the Sui node binary version.

Select `RPC Full Node` -> `Update FullNode Version`

![Sui Wizard FullNode Options](/node-setup/sui-wizard/assets/sui-wizard-node-operations.png)

## Updating Validator Binary Version

If you are running a Sui full node or a Sui validator, you can use Sui Wizard to update the Sui node binary version.

Select `Validator` -> `Update Validator Version`
![Sui Wizard Validator Options](/node-setup/sui-wizard/assets/sui-wizard-validator-node-operations.png)

## Rename Validator

Select `Validator` -> `Perform Validator Operations` -> `Rename Validator`
![Sui Wizard Validator Options](/node-setup/sui-wizard/assets/sui-wizard-validator-operations.png)

## Become Candidate

Select `Validator` -> `Perform Validator Operations` -> `Become Candidate`
![Sui Wizard Validator Options](/node-setup/sui-wizard/assets/sui-wizard-validator-operations.png)

## Join Committee

Select `Validator` -> `Perform Validator Operations` -> `Join Committee`
![Sui Wizard Validator Options](/node-setup/sui-wizard/assets/sui-wizard-validator-operations.png)
